# Sortalizer Feedback Tracker

This repository is the public issue tracker for **Sortalizer.com**.

It exists to collect bugs, feature requests, and product feedback from real use.  
There is no application code in this repository.

## What this tracker is for

Use this tracker to report:

- Bugs or unexpected behavior
- Feature requests and product improvements
- Data quality issues or incorrect identification
- UX friction or confusing flows
- Account support
- Billing or payment issues
Issues submitted through the Sortalizer feedback form are automatically posted here.

## What this tracker is not for

Do not use this tracker for:

- Personal data, private information, or credentials
- Time-sensitive support requests

For private matters, contact:

info@creativemayhem.ltd or see the impresum.

## How issues are handled

- Issues may be edited, relabeled, merged, or closed during triage.
- Not all requests will be implemented.
- Closed does not always mean rejected. It may mean deferred, duplicated, or resolved elsewhere.
- This tracker reflects product direction, not a support SLA.

## Labels

Issues are typically labeled as:

- bug
- feature
- feedback
- from-prod

## Transparency note

This repository is intentionally public.  
Visibility is part of how Sortalizer is built and improved.

If you are uncomfortable with public discussion, do not include sensitive details in your report.
